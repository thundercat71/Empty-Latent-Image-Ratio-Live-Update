Install:
Copy / Git into your ComfyUI custom_nodes folder.

Auto updating Empty Latent Image node that supports Ratios and Landscape and Portrate modes. So if you change either height or width it will auto update the other based on the Ratio and Orientation selected, this will also update live in the ComfyUI window.

As this relies on some extra Java Script it has only been tested at this moment in Chrome and Edge.
